VHS Community - get involved!

VHS is made up of lot of very wonderful humans. The more you connect with the community, the more VHS has to offer.

Here are some ways to connect with the VHS community:

Talk

VHS' central community hub is the web-based forum Talk. This is where longer-lived conversations, information, and announcements go - posts here are available to all VHS members, and some sections are available to the general public!

Join Talk

Slack

For more ephemeral discussions, join the VHS slack - a live chat web app. You can request an invite to Slack here:

Request Slack Invitation

Development

VHS is built on lots of tech. Some custom, some off-the-shelf, all set up and kept running by members. If you're handy with php, node, devops, or similar technologies, and want to help out:

Source code

The code for this web app (Nomos), and most of our other infrastructure, is located on Github:

https://github.com/vhs/nomos

Slack

Check out the #vhs-nomos, #vhs-it-ops, and #vhs-smartspace channels on the VHS slack.
